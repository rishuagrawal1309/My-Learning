Git Stash

Git Stash is used to temporarily save changes that are not yet ready to be committed.

Usage:

  • git stash: Stash the current changes.
  • git stash apply: Apply the most recent stash.
  • git stash list: List all stashed changes.
  • git stash drop: Remove a specific stash.
  1. Save your changes using git stash.
  2. Switch to another branch or perform other operations.
  3. When ready, return to your original branch and apply the stash with git stash apply.
